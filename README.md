# Whispers of the Forest

**Whispers of the Forest** is a 2D narrative-driven game developed using Godot 4 (Mono) and C#.  
This project is created as a bachelor's thesis and demonstrates a dynamic story system influenced by player choices.

---

## 📦 Requirements

Before running the project, make sure you have:

- Godot Engine 4 (Mono version)
- .NET SDK (version 8.0 or compatible)
- Git (for cloning the repository)

---

## 🚀 How to Run

1. Clone the repository:
   git clone https://github.com/your-username/whispers-of-the-forest.git

2. Open the project in **Godot Engine (Mono)**:
   - Launch Godot
   - Click "Import"
   - Select the `project.godot` file

3. Wait for C# build (if required)

4. Run the game:
   - Press **Play** (F5)

---

## 🎮 Controls

- W / A / S / D or ↑ / ← / ↓ / → — move character
- E — interact with objects
- Enter / Space — continue dialogue
- 1–9 — choose dialogue options

---

## 🧠 Project Features

- Dynamic narrative system (Ink + custom logic)
- Story flags system affecting world state
- Reactive dialogues based on player actions
- Interactive objects (trees, props, NPCs)
- Location system without singletons (using LocationContext)

---

## 📁 Project Structure

- scripts/ — core game logic (C#)
- scenes/ — Godot scenes
- data/ — resources (characters, configs)
- addons/ — external plugins (e.g., GodotInk)
